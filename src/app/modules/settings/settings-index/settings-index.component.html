Settings for Metaheuristic

  • Security

    You can change your password here.

  • Tokens

    For using MHBP with LLMs (i.e. ChatGPT) you can specify your tokens, which are provided by API publisher. Name of token can be any but we recommend the following scheme:

    <PROVIDER_NAME>_API_TOKEN , i.e. OPENAI_API_TOKEN

  • Languages

    Here you can change the current language of UI.
